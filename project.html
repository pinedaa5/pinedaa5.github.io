Ransomware

AbstractRansomware is malicious software that encrypts a victim’s files or systems and demands payment to restore access. It spreads via phishing, insecure services, or outdated software, often crippling entire networks. Impacts include data loss, downtime, financial costs, and reputational harm. Defenses include patching systems, offline backups, network segmentation, and phishing awareness training.

Botnets

AbstractA botnet is a network of compromised computers controlled remotely by attackers. Infected devices are used for DDoS attacks, spam, and malware distribution. Botnets evolve by hiding their command servers and using encryption, making detection harder. Prevention includes patching systems, changing default IoT passwords, disabling unnecessary services, and monitoring unusual network traffic.

Malware Detection & Prevention

AbstractMalware detection and prevention rely on layered defenses: strong authentication, software updates, least-privilege access, and backups. Detection has shifted from signature-based methods to behavior and anomaly-based analytics. Organizations also use endpoint detection, logging, and SIEM tools. NIST guidelines stress prevention, monitoring, and incident readiness as part of a holistic strategy.
